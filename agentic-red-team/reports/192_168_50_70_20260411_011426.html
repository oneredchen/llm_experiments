Executive Summary

A penetration test was conducted against the internal host 190.168.50.70 to identify security vulnerabilities. The assessment involved active reconnaissance, port scanning, and service enumeration. The testing identified several critical-risk vulnerabilities, most notably an active root bindshell and a remote code execution vector via distccd. While the assessment concluded without a recorded successful foothold in the final phases, the discovery of highly critical services suggests that the host is significantly compromised at a service level. The presence of anonymous FTP access, exposed system configuration files, and unauthenticated shell access indicates a failure in basic security hardening. Key findings include the existence of a root-level bindshell on port 1524, the presence of an RCE vector on port 3632, and widespread information disclosure through Apache and PHP configurations. Total findings identified include 1 critical, 3 high, 1 medium, and 1 low severity issue. Immediate remediation is required. It is recommended to disable the root bindshell, patch the distccd service, disable anonymous FTP access, and implement strict access controls on SMB shares and web directory indexing.

Critical Overall Rating
